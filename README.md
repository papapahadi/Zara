# Zara Clone

This project is a simple clone of Zara's e-commerce platform built using SwiftUI. It contains several sections such as the **Home page**, **Men section**, **Ladies section**, and **About Us** page. The app is designed to provide an intuitive user experience, showcasing a minimal collection of fashion items for men and women.

## Features

- **Home Page**: The central hub of the app where users can navigate to different sections of the store.
- **Men Section**: Displays two men's clothing items. Users can view details of each item.
- **Ladies Section**: Displays two women's clothing items with the same functionality as the men's section.
- **About Us Page**: Provides information about the brand and its values.

1. Clone this repository to your local machine.
2. Open the project in Xcode.
3. Build and run the app on the simulator or your device.
